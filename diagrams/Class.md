# Глоссарий

1. **Schedule** - класс, представляющий расписание занятий
2. **Group** - класс, представляющий учебную группу
3. **Trainer** - класс, представляющий тренера/преподавателя
4. **Hall** - класс, представляющий зал/аудиторию
5. **Student** - класс, представляющий студента/ученика
6. **Groups_Students** - класс-связка между группами и студентами
